# **App Name**: Churn Station Cinematic

## Core Features:

- Scroll-Synched Visualizer: A high-performance parallax engine mapping 192 WebP frames to scroll position, creating a cinematic forward and reverse motion effect.
- Cloudinary Asset Pipeline: Dynamic fetching of sequence assets from the 'Churn Station Beljium' resource list using the Cloudinary API for optimized delivery.
- AI Flavor Alchemist Tool: An interactive tool that generates bespoke, evocative flavor descriptions and pairing suggestions based on the ingredients list.
- Progressive Asset Loader: A full-screen precision loading overlay with a percentage indicator that ensures all visual sequences are cached before interaction.
- Branch Explorer: An interactive guide to the Islamabad, Rawalpindi, and Lahore locations, featuring late-night hours availability and local contact details.
- Ingredients & Nutrition Vault: Elegant visual cards displaying natural ingredients and styled nutrition labels for premium transparency.
- Unified Feedback Gallery: A dedicated review display showing high-quality testimonials and an aggregate star rating for Churn Station.

## Style Guidelines:

- A luxury Dark Mode palette utilizing a Deep Raspberry Pink (#E0115F) for primary accents and CTA buttons.
- The background is a sophisticated Dark Ebony (#0E0D0D), while Cream White (#F5F5DC) is used for high-contrast typography and subtle dividers.
- An accent color of Rich Orchid (#DA70D6) provides an analogous transition for secondary highlights.
- Headline pairing: 'Playfair' (serif) for bold flavor titles like 'BELGIAN' to evoke classic elegance, paired with 'PT Sans' (sans-serif) for body copy to maintain clarity.
- Full-screen single-page layout featuring a cinematic left-aligned text overlay and social monochrome icons pinned to bottom center.
- Scroll-based animation frames are played through a non-autoplay parallax sequence for total user control over the visual story.
- Minimal, monochrome line icons for social links and interaction points to maintain a high-end, uncluttered agency feel.
